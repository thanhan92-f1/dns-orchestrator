# v1.2.0 发行说明

## 概述

v1.2.0 主要完成了前端路由系统迁移和跨平台架构优化，提升了移动端用户体验，并增强了 Web 端功能支持。

## 新功能

### 移动端导航优化

- **汉堡菜单导航**：第一层页面（域名、账户、设置、工具箱）使用汉堡菜单替代返回按钮，支持快速切换页面
- **MobileMenuTrigger 组件**：封装移动端侧边栏触发逻辑，统一导航体验

### 账户数据清理机制

- **自动清理**：删除账户时自动清理相关的最近访问记录和域名缓存
- **无效数据清理**：应用启动时自动清理无效的最近域名记录

### Web 端功能增强

- **文件操作支持**：Web 端现支持完整的账户导入/导出功能
- **更新检查**：Web 端通过 GitHub Release API 检查更新

## 改进

### React Router 迁移

- **标准路由系统**：从基于组件状态的路由迁移到 React Router v7
- **URL 导航**：支持浏览器地址栏导航和历史记录
- **RootLayout 组件**：集中处理初始化逻辑和全局布局

### 跨平台架构优化

- **文件服务抽象**：`file.service.ts` 统一封装 Tauri 和 Web 平台的文件操作
- **外部链接处理**：`open-external.ts` 统一处理不同平台的外部链接打开
- **平台代码分离**：通过 Vite alias 实现编译时平台切换

### 用户体验优化

- **加载状态**：导入对话框、DNS 记录表单和删除对话框添加加载状态，防止重复提交
- **Safe Area 适配**：优化移动端安全区域支持和滚动区域底部间距
- **对话框动画**：移除不必要的滑动动画，提升响应速度

## 修复

- 修复账户删除后最近域名列表未及时更新的问题
- 修复 Tailwind 无效类名 `text-md` → `text-base`

## 升级说明

从 v1.1.0 升级无需额外操作，直接安装新版本即可。
